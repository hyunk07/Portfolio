    스와이퍼 버튼이 왜 내가 입력한 값으로 안바뀔까?

    스와이퍼 버튼을 조정하면 마지막으로 준 값으로 버튼이 변경되는 사태가 발생. 개발자 도구를 열어 확인해보니 미처 바꾸지 못한 class값이 위쪽의 다른 스라이드에 영향을 주는것을 발견 모든 class값을 다르게 지정하니 해결됨

    스와이퍼 nav 버튼을 변경하고 싶다...

    스와이버 기본 꺽새 모양 버튼을 내가 원하는 모양으로 바꾸고 싶은데 width을 주고 background-color을 주어도 모양이 바뀌지 않아 인터넷에 검색해 보니 background-image을 주는 방법이 있어 변경했지만 꺽새모양은 사라지지않아
    .swiper-button-next:after,
    .swiper-button-prev:after {
    display: none;
    }
    값을 줘서 삭제하여 완성했다.
